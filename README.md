# Task Management App

A comprehensive Task Management application built with Flutter, designed to help gig workers organize their work efficiently.

[![Releases](https://img.shields.io/github/v/release/mustardian/Task-Management-App?style=for-the-badge)](https://github.com/mustardian/Task-Management-App/releases)

## Features

- **User Authentication**: Secure sign-up and login with Email, Google, Facebook, and Apple (UI).
- **Task Management**: Create, read, update, and delete tasks.
- **Organization**: Filter tasks by category/status (Overdue, Today, Tomorrow, This Week, Later) and sort by date.
- **Search**: Integrated search bar to quickly find tasks.
- **Responsive Design**: Clean, modern UI inspired by Google Keep.
- **Cross-Platform**: Built for Android, iOS, and Web.

## Technology Stack

- **Framework**: Flutter
- **Language**: Dart
- **State Management**: Flutter Riverpod
- **Routing**: GoRouter
- **Backend/Database**: Firebase (Auth & Firestore)
- **Design**: Material 3, Font Awesome Icons

## Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) installed.
- Valid `google-services.json` (Android) from Firebase Console placed in `android/app/`.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mustardian/Task-Management-App.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Task-Management-App/task_manager
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
